Feito com Django, Html e CSS.

necessário instalar o django: pip install Django==3.2.9

pillow : pip install --upgrade Pillow


##para rodar 
tem que iniciar pelo terminal, 
ir até a pasta do arquivo manage
e da o comando manage.py runserver
abrir no navegador o localhost
colocar uma imagem e cortar
a imagem vai ficar salva na pasta media


infelizmente fiquei sem tempo pra terminar 100%
