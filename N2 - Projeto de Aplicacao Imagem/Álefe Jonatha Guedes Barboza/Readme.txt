Disciplina: Processamento de imagem

Projeto de processamento de imagem utilizando interface gráfica e implementando as ferramentas: Balanço de Cores, Filtros negativo e cinza, Contrastre, Brilho, Nitidez.

##Requerimentos 
Python >= 3.9.5 
Pillow >= 8.4.0 
opencv-python >= 4.5.3.56

##Instalação
*Se NÃO tiver o python instalado na maquina (win,mac) entrar no site (https://www.python.org/) menu downloads, escolher o SO e a versão.
*pip install pillow
*pip install opencv-python


Agora é só clicar com o botão direito do mouse na área de edição do VS CODE e selecione a opção “Run python file in terminal”.

##Outro metodo para abrir
Abrir arquivo "Conversor.rar" e extrair o executável "Conversor.exe".
