1. INTRODUÇÃO

Este projeto tem como objetivo oportunizar a implementação de componentes de processamento de imagem como atividade de 2 unidade para os alunos de CCO da FPB.

O NODE IMAGE é um sistema de processamento de imagem que deve ser contruído na linguagem de maior desenvoltura do aluno, podendo ser individual ou em dupla. A restrição de linguagem é feita pelas ferramentas de acompanhamento, devendo ser compatíveis com:
> GIT
> Visual Studio Code

Qualquer AddOns deve ser indicado no README.txt do Projeto do Aluno.

O NODE IMAGE é um sistema de processamento de imagem não destrutivo, onde o usuário possui controle e feedback visual em tempo real das modificações realizadas na tela do sistema. O projeto terá um grupo de requisitos COMUM a todos os projetos e outro grupo de requisitos que os alunos poderão optar pela implementação de parte dele. Essa lista será consolidada nas próximas versões deste documento.

1.1 Finalidade
Este projeto, individual ou em dupla, é componente A5 de nota da FPB. A finalidade do NODE IMAGE é ser uma ferramenta de edição de imagens para redes sociais.
